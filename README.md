# Expense Tracker

The Expense Tracker is a Python application built using Tkinter that allows users to manage and track their monthly expenses. Users can input their monthly income, select expense categories, and enter expenses for each category. The program then calculates the total expenses and profit for the month based on the income and expenses entered.

## Features

- Input monthly income
- Select from predefined expense categories
- Enter expenses for each category
- Calculate total expenses and profit for the month
- Display a congratulatory message if the user has made a profit
- Display a warning message if the user has made a loss

## How to Use

1. Run the Python script `expense_tracker.py`
2. Input the monthly income in the provided entry field
3. Select an expense category from the dropdown menu
4. Enter the expense for the selected category
5. Click "Select" to add the expense to the tracker
6. Repeat steps 3-5 for each expense category
7. Click "Submit" to calculate the total expenses and profit for the month
8. Click "Refresh" to start a new expense tracking session

## Requirements

- Python 3.x
- Tkinter library
